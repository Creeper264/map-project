# 危险化学品企业安全管理系统

## 项目简介

本项目是一个基于 Vue 3 + TypeScript + ECharts 构建的危险化学品企业安全管理系统前端应用。系统旨在通过信息化手段提升企业安全管理水平，实现安全风险的数字化、智能化管控。

## 技术栈

- **框架**: Vue 3.3.11 (Composition API)
- **语言**: TypeScript 5.2.2
- **构建工具**: Vite 5.0.8
- **UI组件库**: Ant Design Vue 4.0.7
- **数据可视化**: ECharts 5.4.3 + vue-echarts 6.6.1
- **路由管理**: Vue Router 4.2.5
- **状态管理**: Pinia 2.1.7
- **日期处理**: Day.js 1.11.10
- **HTTP客户端**: Axios 1.6.2

# 危险化学品企业安全管理系统（前端）

简要说明：这是一个基于 Vue 3 + TypeScript + Vite 的前端示例工程，包含仪表盘、基础信息、重大危险源、双重预防、智能巡检、人员定位、用户认证等页面（部分为前端演示逻辑）。

## 目录速览

根目录下的主要文件与目录：

- `index.html` - 应用入口 HTML
- `package.json` - 依赖与脚本
- `vite.config.ts` - Vite 配置（端口 3000，别名 `@` -> `src`）
- `tsconfig.json` - TypeScript 配置
- `public/` - 静态资源（包含 `campus-map.png`）
- `src/` - 源代码：
   - `main.ts` - 应用入口（挂载 Vue、Pinia、路由、Antd）
   - `App.vue` - 根组件
   - `router/index.ts` - 路由定义与路由守卫
   - `stores/user.ts` - Pinia 用户状态（模拟登录/注册/token 持久化）
   - `layouts/MainLayout.vue` - 主布局（侧边栏 + 内容区）
   - `pages/*.vue` - 各业务页面（Login、Register、Dashboard、BasicInfo、MajorHazard、DualPrevention、SpecialWork、IntelligentInspection、PersonnelLocation）

## 关键功能与实现说明

- 路由：`src/router/index.ts` 使用按需加载（组件懒加载）并实现了基于 `stores/user` 的路由守卫，未登录用户会被重定向到 `/login`。
- 状态管理：`src/stores/user.ts` 使用 Pinia，包含 `login`、`register`、`logout`、`checkLogin` 等模拟逻辑，并将 token 与 userInfo 持久化到 `localStorage`。
- 地图与巡检：项目使用 `public/campus-map.png` 作为静态地图背景，用于人员定位与巡检页面的展示（若需真实地图，可替换为高德/百度/Mapbox 等服务）。

## 常用文件说明

- `src/main.ts` - 创建应用并挂载插件（Pinia、Router、Antd）
- `src/App.vue` - 根视图，仅包含 `<router-view />`
- `src/router/index.ts` - 路由表与导航守卫实现
- `src/stores/user.ts` - 用户认证相关的前端状态和持久化逻辑

## 使用指南

### 快速开始

1. **启动项目后，系统会自动跳转到登录页面**

2. **注册新账户**
   - 点击登录页面的"还没有账号？立即注册"链接
   - 填写用户名（3-20个字符）
   - 填写手机号（11位数字）
   - 点击"发送验证码"按钮
   - 查看页面提示消息，获取验证码（演示模式会在消息中显示验证码）
   - 输入6位验证码
   - 设置密码（6-20个字符）
   - 再次确认密码
   - 点击"注册"按钮
   - 注册成功后，会弹出成功提示框，点击确定后自动跳转到系统首页

3. **登录账户**
   - 在登录页面输入用户名和密码
   - 点击"登录"按钮
   - 登录成功后，会弹出欢迎提示框，点击确定后自动跳转到系统首页

4. **使用系统**
   - 登录后可以访问所有功能模块
   - 右上角显示当前登录用户名
   - 点击用户名下拉菜单可以查看个人信息或退出登录

5. **退出登录**
   - 点击右上角用户名
   - 选择"退出登录"
   - 确认后退出并返回登录页面

### 验证码说明

- **演示模式**：当前版本为演示模式，验证码由前端生成并在页面消息提示中显示
- **实际使用**：实际项目中应通过后端API发送短信验证码到用户手机
- **倒计时**：发送验证码后，按钮会显示60秒倒计时，期间无法重复发送
- **验证码格式**：6位数字

### 登录状态保持

- 登录状态会保存在浏览器本地存储（localStorage）中
- 刷新页面后登录状态会自动恢复
- 清除浏览器缓存或退出登录会清除登录状态

### 路由保护

- 未登录用户访问系统页面会自动跳转到登录页
- 已登录用户访问登录/注册页会自动跳转到系统首页
- 登录后会自动跳转到之前尝试访问的页面（如果有）
